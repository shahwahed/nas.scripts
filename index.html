netapp.config.maker.py

This script will create .config and .rc file for Netapp nas filer based on a XML configuration file.

Requirements / Pre-requisites

This script require Python 2.X run on a Linux server, haven't test on windows few python libs (lxml, argparse)

usage

from a linux server :

#shell> netapp.config.maker.py [-h] -c configfile [-v]

Quick help

` usage: netapp.config.maker.py [-h] -c configfile [-v]

.config and .rc file generator based on XML config file for Netapp filer

optional arguments: -h, --help show this help message and exit -c configfile, --configfile configfile XML config file -v, --vfiler vfiler config generator only `
